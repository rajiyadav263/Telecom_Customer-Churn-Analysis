# Customer Churn Analysis Dashboard

## Project Overview
This project focuses on analyzing customer churn behavior using Microsoft Excel. The dashboard provides insights into customer retention, churn trends, revenue impact, and factors influencing customer attrition through interactive visualizations and KPI metrics.

---

## Objectives
- Analyze customer churn patterns
- Identify high-risk customers
- Track revenue loss due to churn
- Understand customer behavior and retention trends
- Create an interactive business dashboard

---

## Tools & Features Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
- Conditional Formatting
- KPI Cards
- Data Cleaning & Transformation

---

## KPI Metrics
- Total Customers
- Churned Customers
- Churn Rate %
- Revenue Lost
- Average Monthly Charges

---

## Dashboard Features
- Churn by Contract Type
- Churn by Payment Method
- Internet Service Analysis
- Tenure Analysis
- Monthly Charges Analysis
- Interactive Filters & Slicers

---

## Key Insights
- Month-to-month contract customers show the highest churn rate.
- Customers with higher monthly charges are more likely to churn.
- Electronic check payment users contribute to higher customer attrition.
- Long-tenure customers are more loyal and less likely to leave.
- Fiber optic internet users show higher churn behavior.
- Frequent support calls indicate higher churn risk.
